# Etch‑a‑Sketch (Browser Grid Drawing App)
An interactive, browser‑based drawing application inspired by the classic Etch‑a‑Sketch.
The user draws directly on a grid using the mouse, with multiple drawing modes, dynamic grid resizing, and instant reset functionality.

## How It Works

The application initializes automatically on page load by creating a square drawing grid.
The grid consists of dynamically generated cells based on the selected grid size.
The user can draw by clicking or hovering over grid cells.
The grid can be resized or cleared at any time using UI buttons.

## How to Use

Open the project in your browser.
Draw on the grid by clicking or moving the mouse over cells.
Click Select grid size to generate a new grid.
Click Clear grid to reset the drawing area.
